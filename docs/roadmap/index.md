# 阅读地图

你可能刚开户，或者还没有开户，对股票、交易规则、估值和技术分析都只是零散听说过。

推荐顺序：

1. [7天学习计划](/plan/7-day)
2. [入门认知与交易机制基础版](/notes/intro-and-rules-basic)
3. [市场结构与不同交易场景基础版](/notes/market-structure-basic)
4. [估值与选股框架基础版](/notes/valuation-and-selection-basic)
5. [方法论与心态基础版](/notes/methodology-and-mindset-basic)

### 2. 已开户但没体系

你可能已经看过盘、买过股票，但规则、估值、技术、仓位这些知识是分散的，脑子里没有一条完整顺序。

推荐顺序：

1. [股票知识总览区](/notes/intro-and-rules)
2. [估值与选股框架进阶版](/notes/valuation-and-selection-advanced)
3. [技术分析导读](/technical/)
4. [K线基础版](/technical/k-line-basic)
5. [均线基础版](/technical/moving-averages-basic)
6. [方法论与心态进阶版](/notes/methodology-and-mindset-advanced)

### 3. 会看盘但容易乱交易

你可能已经会看一些图形、指标和分时，但总是容易追着波动跑，缺少统一框架和纪律。

推荐顺序：

## 常见误区

- 先学分时，再回头补规则
- 先追热点，再补估值
- 先背指标信号，再补方法论
- 学了一堆碎知识，但没有阅读顺序

## 一个简单原则

先学 `规则`，再学 `框架`，然后学 `技术`，最后才是 `执行`。  
如果顺序反了，后面大概率会不断返工。
