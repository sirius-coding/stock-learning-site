# 分时进阶版

## 节奏与结构

分时进阶使用重点，在于理解不同时间框架下的结构关系，例如 5 分钟、15 分钟、30 分钟、60 分钟图与日线之间的配合。

## 进阶观察点

- 60 分钟图是否与日线方向一致
- 分时 V 型反转是否出现在关键位置
- 短线筛选时，量比、换手、流通市值和分时新高是否协同

## 进阶用法

- 用 60 分钟图看短线节奏，用日线定大方向。
- 盘中执行时先找“强背景下的强分时”，而不是只追波动。
- 把分时当执行工具，而不是情绪放大器。

## 风险边界

- 分时结构成功率高度依赖大环境
- 短线系统离不开纪律和止损
- 没有仓位控制时，分时工具会放大错误
