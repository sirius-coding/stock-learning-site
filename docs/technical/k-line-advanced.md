# K线进阶版

## 位置与组合

同样一根K线，出现在上涨尾端、下跌尾端、横盘中部，含义完全不同。进阶使用重点在“位置 + 组合 + 量价”三者结合。

## 进阶观察点

- 看涨/看跌组合是否出现在关键支撑阻力位
- 连续K线结构是否在表达趋势加速或衰竭
- K线形态是否得到量能确认

## 进阶用法

- 用K线组合去观察趋势阶段，而不是追求精确买卖点。
- 对经典形态先问“为什么出现在这里”，再问“像不像教材里的图形”。
- 结合骗线思路，区分真实突破和被刻意制造的形态。

## 风险边界

- 纯K线分析容易被主力资金扰动
- 震荡市里形态成功率显著下降
- 没有周期和仓位约束时，进阶形态也会失真
