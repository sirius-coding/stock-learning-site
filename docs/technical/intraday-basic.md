# 分时基础版

## 概念

分时图描述的是更短周期下的即时价格变化，更适合做执行层观察而不是主逻辑判断。

## 常见信号

- 白线强于黄线：个股强于均价表现
- 分时创新高：当天强势延续参考
- 回落后快速收复：短线承接较强

## 基础用法

- 先看日线背景，再用分时找执行位置。
- 分时图适合辅助做进出节奏，不适合决定长期方向。
- 与量比、换手率、盘口强弱配合看。

## 常见误区

- 只盯分时，不看日线
- 把一分钟波动当趋势变化
- 没有计划却频繁跟随分时动作
