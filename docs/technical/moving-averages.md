# 均线

## 概念

均线是一定周期内平均价格形成的曲线，常用于观察趋势、强弱和支撑阻力。常见的有 5 日、10 日、20 日等。

## 信号

- 多头排列：整体趋势偏强
- 空头排列：整体趋势偏弱
- 金叉/死叉：均线关系变化的提示信号
- 回踩关键均线：可用于观察趋势是否延续

## 用法

- 5日线适合看短线节奏。
- 20日线适合看中短期趋势和回调强弱。
- 均线最好和量价关系一起看，而不是独立判断。
- 用均线做趋势过滤，比拿它硬找精确买卖点更稳。

## 风险

- 把金叉死叉当自动买卖按钮。
- 忽视大趋势，只盯一两根短周期均线。
- 在震荡市里过度依赖均线信号。
