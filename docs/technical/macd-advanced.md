# MACD进阶版

## 背离与钝化

MACD 进阶使用的重点，在于理解背离、钝化和趋势延续之间的关系。背离是线索，不是自动反转命令。

## 进阶观察点

- 顶背离是否出现在高位压力区
- 底背离是否出现在下跌末端且有量价配合
- 强趋势里MACD是否出现钝化但价格仍持续运行

## 进阶用法

- 把背离作为“需要提高警惕”的信号，而不是立刻反手的信号。
- 用MACD识别趋势衰竭，而不是预测每个拐点。
- 强势趋势里更重节奏修正，弱势环境里更重风险控制。

## 风险边界

- 背离可以持续很久
- MACD钝化不等于趋势结束
- 没有位置、量能和周期配合时，误判很多
