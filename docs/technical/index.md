# 技术分析导读

## 这组内容怎么读

技术分析更适合用来辅助观察趋势、位置和执行，而不是替代企业分析和估值框架。建议阅读顺序是：

1. `K线`
2. `成交量`
3. `均线`
4. `MACD`
5. `布林线`
6. `分时`
7. `筹码`
8. `盘口`

## 为什么这样排

- 先看 `K线`，因为它是最直观的价格语言。
- 再看 `成交量`，因为量价要配合。
- `均线 / MACD / 布林线` 属于趋势和节奏工具。
- `分时 / 筹码 / 盘口` 更偏执行层观察。

## 使用原则

- 不神化单一指标
- 大周期优先于小周期
- 先有逻辑，再用技术图形找位置
- 技术分析用于辅助，不用于代替风险管理
