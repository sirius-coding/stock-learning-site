# 盘口

## 概念

盘口指标描述的是当日交易状态，主要帮助观察即时活跃度、买卖力量和价格波动特征。

## 信号

- `量比`：相对活跃程度
- `换手率`：流通筹码交换速度
- `委比`：买卖委托强弱参考
- `内盘 / 外盘`：主动性成交方向的辅助信息

## 用法

- 作为当日状态观察工具，判断个股是否活跃、是否有明显资金参与。
- 和分时图、K线位置配合使用，做执行层辅助。
- 适合短线观察，不适合长期价值判断。

## 风险

- 把盘口指标当作长期选股依据。
- 只看单个盘口数字，不看整体结构。
- 过度解读当日噪音。
