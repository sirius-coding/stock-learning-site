# K线

## 概念

K线是一定周期内价格变化的压缩表达，展示开盘价、收盘价、最高价和最低价。实体、上影线、下影线分别对应买卖双方在这个周期里的博弈结果。

## 信号

- 单根K线可以提示情绪强弱，但独立信号很有限。
- K线组合比单根K线更有参考价值。
- 看涨/看跌组合的核心，是判断多空力量有没有发生切换。

## 用法

- 先看K线所处位置，再看形态。
- 与成交量配合，判断上涨/下跌的力量是否真实。
- 用K线识别趋势延续、整理或反转尝试。

## 风险

- 只背形态名称，不理解位置和量价关系。
- 把单根K线当成买卖按钮。
- 忽略主力骗线和短期噪音。
