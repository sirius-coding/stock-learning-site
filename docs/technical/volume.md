# 成交量

## 概念

成交量代表一定时间内交易的活跃程度和多空博弈强度。价格变化背后，往往需要成交量来确认力量是否充分。

## 信号

- 放量上涨：趋势可能得到强化
- 缩量上涨：上涨力度可能不足
- 放量下跌：抛压释放明显
- 恐慌性放量：有时接近阶段性极端

## 用法

- 看K线组合时同步看量，判断形态是否“有力”。
- 看趋势突破时，量能通常是有效性的重要确认条件。
- 用量价关系辅助判断顶部、底部和延续性。

## 风险

- 只看价格不看量。
- 看到放量就默认看多。
- 不区分不同位置的放量含义。
