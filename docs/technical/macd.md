# MACD

## 概念

MACD 是常见的趋势类指标，主要由黄白线、红绿柱组成，用来观察趋势变化、动能强弱以及背离情况。

## 信号

- 金叉/死叉：趋势变化提示
- 红柱变长/变短：多头动能增强/减弱
- 绿柱变长/变短：空头动能增强/减弱
- 顶背离/底背离：价格与动能出现不一致

## 用法

- 先拿MACD看趋势和节奏，不要先拿它抄底逃顶。
- 用背离当作线索，再结合价格位置和成交量做判断。
- MACD更适合辅助，而不是独立开仓。

## 风险

- 见背离就立刻逆势操作。
- 不区分趋势市和震荡市。
- 把MACD当成万能预测工具。
