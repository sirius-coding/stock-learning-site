# K线基础版

## 概念

K线用一个周期内的开盘、收盘、最高、最低四个价格来表达市场行为。对新手来说，先学会读实体、上影线和下影线的含义，比背形态名字更重要。

## 常见信号

- 长阳线：多头阶段性占优
- 长阴线：空头阶段性占优
- 长上影线：上方抛压重
- 长下影线：下方承接存在
- 十字星：多空短暂平衡

## 基础用法

- 先看方向，再看强弱。
- 先看所处位置，再看形态名称。
- 先结合成交量，再决定信号是否值得重视。

## 常见误区

- 看到单根K线就下结论
- 不管位置，只记模板
- 脱离趋势判断单独操作
