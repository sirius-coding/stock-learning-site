# 筹码

## 概念

筹码分布用来观察市场持仓成本区、套牢盘分布和筹码集中/发散状态，更适合做位置判断和压力支撑观察。

## 信号

- 筹码集中：成本区相对明确
- 筹码发散：分歧更大
- 上方密集筹码：潜在套牢压力
- 下方密集筹码：潜在支撑区域

## 用法

- 观察价格处于主要成本区上方还是下方。
- 配合趋势判断，理解压力位与支撑位的强弱。
- 用于辅助识别风险收益比，而不是单独开仓。

## 风险

- 脱离趋势单独看筹码。
- 把筹码区当成绝对支撑或绝对压力。
- 忽视筹码结构会随着时间变化。
