# 布林线

## 概念

布林线由上轨、中轨、下轨组成，用来描述价格波动区间、趋势方向和支撑压力位置。

## 信号

- 上轨附近：常见压力观察区
- 下轨附近：常见支撑观察区
- 向上开口：趋势偏强
- 向下开口：趋势偏弱
- 喇叭口变化：波动结构可能在切换

## 用法

- 用布林线看价格所处区间，而不是机械地“碰上轨就卖、碰下轨就买”。
- 中轨可辅助理解趋势强弱。
- 结合量价和位置，判断超买超卖是短期修复还是趋势反转。

## 风险

- 把下轨当无条件买点。
- 忽视强趋势里价格沿轨运行的可能性。
- 单独使用布林线判断趋势反转。
