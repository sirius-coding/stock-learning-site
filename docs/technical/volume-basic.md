# 成交量基础版

## 概念

成交量反映市场活跃度和多空博弈强度。看价格时同步看量，能帮助你判断动作是否“有力”。

## 常见信号

- 放量上涨：多头力量增强
- 缩量上涨：上行动能不足
- 放量下跌：抛压加重
- 缩量下跌：卖压阶段性减弱

## 基础用法

- 先判断价格方向，再看量是否匹配。
- 突破时看量，回调时也看量。
- 量价背离时提高警惕。

## 常见误区

- 只看价格不看量
- 把所有放量都理解成利好
- 不区分高位放量和低位放量
