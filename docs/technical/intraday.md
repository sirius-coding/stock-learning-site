# 分时

## 概念

分时图反映的是更短周期下的即时强弱，是执行层工具，不是替代日线逻辑的主框架。

## 信号

- 白线强于黄线：个股当日更强
- 分时创新高：短线强势延续的参考
- V 型反转：短线节奏变化的表现
- 60 分钟图：短线观察里兼顾稳定性与灵敏度

## 用法

- 先看日线或更大周期的背景，再用分时找执行位置。
- 短线筛选时可配合量比、换手率、流通市值和分时强弱。
- 分时更适合确认当天资金强弱，而不是决定中期方向。

## 风险

- 只盯分时图，被噪音带走。
- 把分时小波动误判成趋势反转。
- 没有日线背景就做超短判断。
