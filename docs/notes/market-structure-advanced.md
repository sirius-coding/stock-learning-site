# 市场结构与不同交易场景进阶版

## 边界条件

市场结构不是为了背诵规则表，而是为了判断某个机会是否适合你的方法和风险承受能力。

## 与其他主题的关系

- 与 `估值` 的关系：不同市场对同类资产的估值习惯可能不同。
- 与 `技术分析` 的关系：同样的图形，在不同市场里成功率和波动后果可能不同。
- 与 `方法论` 的关系：方法有效性常常依赖市场环境。

## 组合判断方式

- 先识别市场类型
- 再判断这个市场里的波动、流动性和成本是否匹配你的策略
- 最后再决定是否进入具体标的分析

## 容易误判的场景

- 只看涨跌幅，不看制度约束
- 把高弹性市场误认为更容易赚钱
- 忽视港股通中的汇率、费用和交收差异
