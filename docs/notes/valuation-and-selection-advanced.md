# 估值与选股框架进阶版

## 边界条件

估值难点不在公式，而在于你是否理解价值来自哪里，以及哪些企业根本不该用同一把尺子衡量。

## 与其他主题的关系

- 与 `市场结构` 的关系：不同市场会放大或压制某些估值风格。
- 与 `方法论` 的关系：没有仓位和时机的估值判断是不完整的。
- 与 `技术分析` 的关系：技术图形最多帮你找位置，不能代替估值框架。

## 组合判断方式

- 先判断行业和商业模式
- 再选更合适的估值工具
- 再把估值和预期、周期、仓位一起看

## 容易误判的场景

- 把成长假设当事实
- 用静态估值看强周期企业
- 选对公司却在错误的预期区间买入
