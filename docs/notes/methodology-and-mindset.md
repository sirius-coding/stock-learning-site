# 方法论与心态

## 主题说明

真正决定新手能不能走远的，往往不是会不会看图，而是有没有正确的顺序、预期、纪律和自我校正能力。

## 关键概念

- 股市长期获胜的是少数人
- 散户也有灵活性优势
- 投资本质是前瞻
- 周期、预期、认知偏差会持续影响决策

## 详细笔记

### 散户先承认难度

- 市场不是提款机，也不是纯随机游戏。
- 先接受“长期稳定获胜很难”，才能减少冲动。

### 散户也有优势

- 灵活、规模小、没有机构的仓位约束，可以等待更久、更专注。

### 决策链比单个结论更重要

- 认知偏差会污染整个判断过程。
- 投资问题最终都要回到：买什么、何时买、买多少。

### 周期和预期

- 周期不是例外，是常态。
- 市场价格不只反映事实，也反映预期与预期差。

### 最小可执行顺序

1. 这是什么公司
2. 它处在什么市场和行业环境
3. 它大概贵不贵
4. 趋势和位置如何
5. 买多少
6. 错了怎么退

## 易混点

- 还没建立体系就高频操作
- 以为看懂一个指标就能稳定赚钱
- 看对了对象却买贵了、买少了、拿不住
