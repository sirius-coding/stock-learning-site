# 入门认知与交易机制

## 主题说明

股票不是单纯的价格波动工具，而是企业所有权的一部分。新手先搞清交易对象和市场规则，比提前学“战法”更重要。

## 关键概念

- 股票代表企业权益
- A股交易有明确制度约束
- 委托和撮合规则会直接影响成交结果
- 新股申购、分红、集合竞价都属于基础机制的一部分

## 详细笔记

### 先理解你买的是什么

- 买股票本质上是买企业未来盈利与市场定价的组合。
- 如果只把股票看成一串代码，就容易把所有操作退化成追涨杀跌。

### 交易规则要先于判断

- 新手至少要先掌握 `T+1`、涨跌停、交易时段、最小买卖单位。
- 很多亏损并不是分析错，而是对制度不了解。

### 委托方式直接影响执行

- `限价委托` 更适合新手，因为它先控制价格。
- 市价委托强调成交优先，但在波动时更容易出现偏离预期的成交价格。

### 集合竞价和分红不要误读

- 集合竞价是开盘前的集中撮合，不是随机定价。
- 股票分红后通常会除权除息，不能把分红机械理解成“白赚”。

## 易混点

- 分红不是额外送钱
- 集合竞价不是万能开盘信号
- 不懂委托方式时，不要急着用市价单
