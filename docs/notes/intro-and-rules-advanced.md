# 入门认知与交易机制进阶版

## 边界条件

入门规则看起来简单，但真正的难点在于：规则会改变你“以为自己能做”的很多动作。比如你知道 `T+1`，但未必真正理解它对追涨、止损和隔夜风险管理的限制。

## 与其他主题的关系

- 与 `市场结构` 的关系：不同市场规则不同，基础规则不能机械平移。
- 与 `盘中工具` 的关系：很多盘中信号只有放在规则约束下才有意义。
- 与 `方法论` 的关系：规则意识是纪律的最底层。

## 组合判断方式

- 先问：这个动作在制度上可不可行？
- 再问：这个动作在成本上值不值得？
- 最后问：这个动作在风险上是否可承受？

## 容易误判的场景

- 把集合竞价信号当成全天方向
- 把分红、送股当成天然利好
- 不理解委托和成交机制造成的滑点与偏差

## 进阶提醒

真正成熟的交易者，往往最先尊重的不是观点，而是规则。
