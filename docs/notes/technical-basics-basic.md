# 技术分析基础版

## 定义

技术分析是对价格、成交量和常见指标的结构化观察，用来辅助判断趋势、强弱和执行位置。

## 核心框架

- K线看结果
- 成交量看力量
- 均线看趋势
- MACD 和布林线看节奏与区间

## 最小必知点

- 单一指标不可靠
- 大周期优先于小周期
- 价格位置和量能要一起看

## 常见误区

- 学一个指标就想实盘
- 只背信号名词
- 不看位置和市场环境
