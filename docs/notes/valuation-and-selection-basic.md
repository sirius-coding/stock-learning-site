# 估值与选股框架基础版

## 定义

这一组内容回答两个最基本的问题：什么叫估值，什么叫值得研究的公司。

## 核心框架

- `PE / PB / PS / PCF / PEG` 是常见表达工具
- 没有万能估值指标
- 选股不是只看便宜，而是看 `好生意 + 好企业 + 好价格`

## 最小必知点

- PE 看盈利
- PB 看净资产
- PS 看收入
- PEG 把成长纳入估值
- 低估值不等于便宜

## 常见误区

- 只会看PE
- 把低估值当成安全垫本身
- 不区分行业差异和公司阶段
