# 盘中工具与短线观察进阶版

## 边界条件

盘中工具越细，越容易诱发情绪化。进阶使用的关键不是看更多细节，而是知道哪些细节值得忽略。

## 与其他主题的关系

- 与 `技术分析` 的关系：这是技术分析在更短周期下的执行层工具。
- 与 `交易机制` 的关系：规则会限制你如何使用盘中信号。
- 与 `方法论` 的关系：没有纪律时，盘中工具会放大错误。

## 组合判断方式

- 先看日线/趋势背景
- 再看盘中强弱是否支持原判断
- 最后才考虑执行，而不是反过来

## 容易误判的场景

- 把盘中噪音误判成趋势反转
- 高量比就默认强势
- 筹码、盘口、分时三个工具给出冲突信号时强行下注
