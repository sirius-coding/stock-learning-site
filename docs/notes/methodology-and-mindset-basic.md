# 方法论与心态基础版

## 定义

这一组内容讲的是：即使你学了规则、估值和技术，最终能不能执行出来，仍然取决于方法论和心态。

## 核心框架

- 市场长期获胜者是少数
- 散户有灵活性优势
- 投资至少要回答：买什么、何时买、买多少

## 最小必知点

- 周期必须被承认
- 认知偏差会污染判断链
- 方法比单次结论更重要

## 常见误区

- 没有体系就频繁操作
- 以为会看盘就等于会投资
- 情绪驱动仓位变化
