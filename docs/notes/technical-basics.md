# 技术分析基础

## 主题说明

技术分析适合用来辅助观察趋势、位置和执行，但不该被神化成预测机器。新手先学骨架，不要一上来就沉迷形态名称。

## 关键概念

- K线是结果，成交量是力量
- 均线看趋势和强弱
- MACD看趋势变化和背离
- 布林线看区间、波动和支撑压力

## 详细笔记

### K线与成交量

- 单根K线意义有限，要结合位置、前后关系和成交量看。
- 成交量代表多空博弈强度，只看形态不看量，判断会失真。

### 均线

- 均线最核心的用途是看趋势、看强弱、找支撑阻力。
- 多头排列、空头排列、金叉、死叉都只是语言，不是自动信号。

### MACD

- MACD可以辅助判断趋势变化、黄白线关系、红绿柱变化和背离。
- 背离是线索，不是单独买点或卖点。

### 布林线

- 布林线上下轨更适合看波动边界和压力支撑。
- 跌破下轨不一定代表反转，更可能只是超卖后的短期反弹。

## 易混点

- 把金叉死叉当买卖按钮
- 看到背离就抄底逃顶
- 只用一个指标决定全部操作
