# 盘中工具与短线观察

## 主题说明

盘口、分时、筹码分布和短线战法，更适合做执行层观察，不适合代替中期判断逻辑。它们的价值在于辅助，不在于单独决定买卖。

## 关键概念

- 盘口指标只能描述当日交易状态
- 分时图适合做执行，不适合替代日线逻辑
- 筹码分布更适合看成本区、压力区和集中度

## 详细笔记

### 盘口指标

- 常见指标包括 `量比`、`换手率`、`委比`、`内盘`、`外盘`、`振幅`、`动态市盈率`。
- 它们是当日状态工具，不是长期价值判断工具。

### 分时图

- 分时图反映即时强弱，适合做买卖执行辅助。
- 但站内内容反复强调：`K线图为主，分时图为次`。

### 筹码分布

- 更适合判断套牢盘、密集成交区和支撑压力区。
- 不能脱离趋势单独使用。

### 短线方法的共同点

- 量比、换手、流通市值、均线排列、分时强弱、放量与否，是常见的短线筛选条件。
- 真正决定结果的仍然是仓位管理和止损纪律。

## 易混点

- 只盯分时图
- 只学买点不学卖点
- 战法一学就重仓实盘
