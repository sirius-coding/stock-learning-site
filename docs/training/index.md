# 一个散户的自我修养

## 这组内容讲什么

这是一条从“股市到底是什么”一路走到“如何形成自己的投资框架”的连续阅读路径。它不是单纯讲技术指标，也不是只讲价值投资，而是把市场事实、认知偏差、企业价值、市场定价、周期和仓位管理串成一套思考顺序。

## 建议阅读方式

1. 按章节顺序读，不建议跳着读。
2. 每章先看核心观点，再看详细摘要。
3. 每读完 2-3 章，回头整理一次自己的观察框架。

## 阅读地图

- `序言，前言`：建立阅读预期
- `第1章-第5章`：先纠正市场认知和思维方式
- `第6章-第10章`：进入价值、企业与风险
- `第11章-第17章`：把对象、时机、仓位、估值和市场串成体系

## 最适合谁

- 已经开户，但总觉得自己在“瞎看盘”的人
- 学过一些技术分析，但没有整体框架的人
- 想把“选股、买点、仓位、心态”连起来的人
