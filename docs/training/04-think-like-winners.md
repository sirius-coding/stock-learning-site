# 第4章 像胜出者一样思考

## 本章主题

学习顶级投资者的共同思维方式，而不是只记他们的口号。

## 核心观点

- 大师的方法不同，但底层原则常常一致
- 长期赢家重视框架，不迷信单一招式
- 思维方式比技巧更能复制

## 详细摘要

这一章强调，不同投资大师的持仓风格和操作方式差别很大，但他们在安全边际、长期主义、能力圈、独立思考等原则上高度一致。新手要学的不是模仿某位大师买了什么，而是理解他们为什么这样思考。

## 新手最该记住的点

- 不要只摘金句
- 先学原则，再谈方法细节
