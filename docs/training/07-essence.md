# 第7章 让视角回归本质

## 本章主题

从表面指标退一步，看市场、商业和投资的本质。

## 核心观点

- 市场的本质是供需
- 商业的本质是竞争
- 投资的本质是前瞻

## 详细摘要

作者在这一章强调，仅仅会背金融概念并不能自动形成判断力。真正的判断力来自于能不能把复杂现象还原到更本质的层面。理解供需、竞争和前瞻，能让人少被故事和情绪带着走。

## 新手最该记住的点

- 看市场先看供需，不只看情绪
- 看公司先看竞争，不只看口号
