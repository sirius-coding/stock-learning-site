# 第11章 对象、时机、力度

## 本章主题

投资不是只解决“买什么”，而是同时解决对象、时机和仓位。

## 核心观点

- 任何成熟策略都要回答三件事
- 投资结果 = 对象 × 时机 × 力度
- 选对公司但买错时点或仓位，结果仍可能很差

## 详细摘要

这一章把投资简化成最实用的三维框架：对象、时机、力度。作者给出公式：净值变化可以拆成每股收益、估值和股票数量。也就是说，最好的投资同时要求企业盈利增长、估值买得合理、仓位配置正确。

## 新手最该记住的点

- 没有仓位问题的“选股策略”是不完整的
- 买得对和拿得住，都离不开前面的理解
