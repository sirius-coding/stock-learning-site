# 第3章 捅破投资的“窗户纸”

## 本章主题

从“进股市”转向“像投资者那样行动”。

## 核心观点

- 真正的投资行为和投机冲动不同
- 投资要有原则
- 先避坑，再谈升级

## 详细摘要

作者把投资者入门拆成四件事：理解什么是投资、知道必须遵守哪些原则、识别常见陷阱、以及如何循序渐进提高自己。它的重点在于提醒新手，投资不是靠激情和运气临时拼出来的，而是一种有边界、有纪律的长期行为。

## 新手最该记住的点

- 没有原则的操作不叫策略
- 会避坑，比会找机会更重要
