# 第13章 预期与回报

## 本章主题

回报不是由“好公司”三个字自动带来的，而是由预期差和风险收益结构共同决定。

## 核心观点

- 回报受多种因素共同影响
- 高回报通常伴随更高挑战
- 股票共性比单个故事更值得研究

## 详细摘要

这一章关注“什么样的股票更容易产生高回报”。作者提醒，回报不是只由企业质量决定，还与市场预期、买入价格、未来兑现路径等密切相关。理解回报机制，能帮助新手少追逐表面上最热的东西。

## 新手最该记住的点

- 好公司不等于好回报
- 预期差往往比故事更重要
