# 总结篇 好生意，好企业，好投资

## 本章主题

把整本书的框架收束成一句话：好生意，好企业，好投资。

## 核心观点

- 好生意决定上限
- 好企业决定兑现能力
- 好投资决定最终回报质量

## 详细摘要

总结篇把全书的主线完整收束起来。作者强调，投资的首要任务是识别好生意，其次是找到能把好生意转化为真实现金流和竞争优势的好企业，最后才是在合适时机、合适价格、合适仓位下把它变成好投资。这里最重要的不是记住一句口号，而是理解三者之间缺一不可。

## 新手最该记住的点

- 只会选公司，不等于会投资
- 眼力比勇气更重要
- 顺序应该是：生意 -> 企业 -> 价格 -> 仓位 -> 持有
