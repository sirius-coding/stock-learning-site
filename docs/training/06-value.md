# 第6章 揭开价值的面纱

## 本章主题

价值投资的起点，不是估值表格，而是先回答“价值到底是什么”。

## 核心观点

- 价值是估值讨论的原点
- 不理解价值，就很难正确估值
- 企业价值最终要落回经营和现金流

## 详细摘要

这一章把读者从价格表面拉回到企业价值本身。无论是市盈率、市净率还是其他指标，都是围绕企业价值展开的工具。如果不先理解企业为什么值钱、价值从哪里来，后面的所有估值计算都容易流于表面。

## 新手最该记住的点

- 价格不是价值本身
- 估值工具只是表达，不是价值来源
